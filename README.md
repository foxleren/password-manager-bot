# Run

To run application you need:
- .env file

## Env configuration

To run API locally you should create your own .env file in the ```backend``` directory  
Example `.env`:

```env
DB_PASSWORD=<password>
BOT_TOKEN=<token>
```

## Local run

```
> make migrate-up
> make run
```
